Native Apps

  • 針對一種裝置開發
  • 簡單的使用到手機的各種功能 (ex: 相機, GPS, 連絡人)
  • 簡單偵測使用者手勢
  • 可離線
  • 從 App Store 下載

PlzHelp 原先的開發方式

Responsive Website

  • 畫面配合裝置視窗寬長, 顯示適合的畫面
  • 需要上網
  • demo

Mobile Web Apps

  • 畫面 Native-like
  • 需要上網, 把使用者導向專門給手機的網頁
  • 可用 JavaScript 去 使用/呼叫 手機功能
上面兩者幾乎相同, 差別是 Mobile Web Apps 的網頁只給手機專用, 可是 RWD 是全部裝置都支援

Hybird Apps

  • Native-like (rendered in a browser)
  • 不同裝置顯示同樣畫面(cross-platform)
  • 使用工具把 Mobile Web 包成像是 Native App
  • 可離線 (限制較多)
  • 從 App Store 下載

PhoneGap is a free and open source framework that allows you to create mobile apps using standardized web APIs for the platforms you care about.

HTML Meta Tag

Android 

<meta name="viewport" content="user-scalable=no,
        initial-scale=1.0, maximum-scale=1, minimum-scale=1,
        width=device-width, height=device-height" />

ios 

<meta name="viewport" content="user-scalable=no,
        initial-scale=1.0, maximum-scale=1, minimum-scale=1,
        width=window.innerWidth, height=window.innerHeight" />

absolute 使得畫面撐大不滑動 

<style>
    position: absolute;
    top: 0;
    bottom: 0;
    left: 0;
    right: 0;
    </style>

media queries 處理不同 device-pixel-ratio 下畫面的顯示 

<style>
    @mixin retina-to($media-Ratina: 1.5) {
    @if ($media-Ratina == 2) {
    @media only screen and (-webkit-min-device-pixel-ratio: 2),
        only screen and (-webkit-device-pixel-ratio: 2) {  @content; }
    }
    @else if ($media-Ratina == 1.5) {
    @media only screen and (max--moz-device-pixel-ratio: 1.5),
        only screen and (-webkit-min-device-pixel-ratio: 1)
                and (-webkit-max-device-pixel-ratio: 1.9),
        only screen and (-webkit-device-pixel-ratio: 1.5),
        only screen and (max-device-pixel-ratio: 1.5) {  @content; }
    }
</style>

使用 Font Icon 取代 png icon

縮放自如, 放大不失真 

<style>font-size: 3em;</style>

變更 icon 顏色 

<style>color: #5000ff;</style>

用 CSS3 transition 取代 jQuery animation

jQuery
$("#start").click(function(){
  $("div").animate({
    opacity: 0.5,
    height:"25px",
  } , 2000);
});
css3
div {
    opacity:1;
    height:20px;
}
div.click {
    opacity:0.5px;
    height:25px;
    transition: opacity 2s, height 2s;
}
